# 🎂 Página Web de Cumpleaños – 22 Años 💖

Este proyecto es una página web especial creada con mucho cariño para celebrar los **22 años de mi novia Shirley** 🥰, de parte de su churris.

## ✨ Descripción

La web funciona como una experiencia interactiva tipo historia, donde se combinan:

* 📸 Recuerdos en forma de imágenes
* 🎥 Videos cortos significativos
* 🎶 Música suave y emotiva
* 💌 Mensajes llenos de cariño

La idea es que al recorrer la página, se sienta como un pequeño viaje por momentos especiales, emociones y recuerdos juntos.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Animaciones y efectos visuales

## ❤️ Objetivo

Más que un proyecto técnico, esta página es un detalle emocional. Representa tiempo, dedicación y sentimientos convertidos en código, hecho especialmente para ella.
